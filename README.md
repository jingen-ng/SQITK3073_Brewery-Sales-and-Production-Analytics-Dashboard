# SQITK3073_Brewery-Sales-and-Production-Analytics-Dashboard
Brewery Sales &amp; Production Analytics Dashboard built with Streamlit to analyze sales, production performance, and operational losses. It provides descriptive analytics, revenue forecasting using Exponential Smoothing, and strategic recommendations to support data-driven decision-making and business planning.

The dataset used in this project is sourced from Hugging Face (Brewery Sales Dataset by Mathi65xl): https://huggingface.co/datasets/Mathi65xl/Brewery_sales and contains historical brewery sales and production data used for analysis and forecasting. 

To run this project, download all files from the repository and ensure they are placed in the same folder. If the files are in a zip format, extract them first. Next, install the required dependencies using the command pip install -r requirements.txt. Once the setup is completed, run the application by executing streamlit run dashboard.py. Make sure all datasets and project files are correctly located in the project directory before starting the app to ensure it runs properly.
